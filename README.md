# landing

## 介绍
使用 [umi脚手架](https://umijs.org/) 快速生成的 react 应用
## 组件库首页
使用 [Landing](https://github.com/ant-design/ant-design-landing) 搭建的页面模版

## 本地开发
``` 
本地服务
npm run start
本地构建
npm run build
```
